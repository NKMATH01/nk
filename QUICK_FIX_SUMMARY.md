# 🚨 빠른 수정 가이드 - NK 학원 업무 관리 시스템

## 발견된 오류 (4개)

### ⭐⭐⭐⭐⭐ 1. 날짜/시간 처리 오류 (가장 중요!)
**증상:** 업무 시간이 9시간 차이로 잘못 기록됨

**원인:** UTC 메서드와 KST 변환이 섞여서 사용됨

**✅ 수정 완료:** Code.gs 34-60줄 수정
- `getUTCFullYear()` → `getFullYear()`
- `getUTCMonth()` → `getMonth()`
- `getUTCDate()` → `getDate()`

---

### ⭐⭐⭐⭐⭐ 2. 벌금 체크 오류
**증상:** 오후 2:30 이후 업무 할당 시 벌금이 잘못 부과됨

**원인:** KST 시간을 UTC 메서드로 체크

**✅ 수정 완료:** Code.gs 326-332줄 수정
- `getUTCDay()` → `getDay()`
- `getUTCHours()` → `getHours()`

---

### ⭐⭐⭐⭐ 3. 마감 시간 비교 오류
**증상:** 업무 완료 시 지연 여부가 잘못 판단됨

**원인:** deadline을 UTC로 해석

**✅ 수정 완료:** Code.gs 391줄 수정
- `+ 'Z'` 제거하여 KST로 해석

---

### ⭐⭐⭐ 4. 건의사항 컬럼 불일치
**증상:** 건의사항 답변 시 오류 가능

**원인:** 6개 컬럼 구조인데 4개만 추가

**✅ 수정 완료:** Code.gs 521줄, 535-544줄 수정
- Reply, ReplyTime 컬럼 추가

---

## 🚀 지금 바로 해야 할 일

### 1단계: 수정된 Code.gs 적용
```
1. 이 폴더의 Code.gs 파일 열기
2. 전체 내용 복사 (Ctrl+A, Ctrl+C)
3. Google Apps Script 편집기 열기
4. Code.gs의 기존 내용 전체 선택 후 붙여넣기
5. 저장 (Ctrl+S)
```

### 2단계: 샘플 데이터 생성 (처음이라면)
```
1. Apps Script 편집기에서
2. 함수 선택: createSampleUsers
3. 실행 버튼 클릭
4. 권한 승인
```

### 3단계: 웹 앱 배포
```
1. 배포 → 새 배포
2. 유형: 웹 앱
3. 액세스 권한: "나"
4. 배포 클릭
5. URL 복사하여 테스트
```

## ⚠️ 만약 여전히 오류가 발생한다면

### 증상: "Users 데이터 없음" 알림
→ `createSampleUsers()` 함수를 실행하세요

### 증상: 시간이 여전히 이상함
→ Apps Script 시간대 설정 확인:
  1. 프로젝트 설정 (⚙️)
  2. 시간대: (GMT+09:00) Asia/Seoul 확인

### 증상: 건의사항/공지사항 오류
→ 해당 시트를 수동으로 삭제하고 다시 실행
