# NK 약술논술 관리 프로그램

안산 NK학원 약술형 논술 대비 학습관리 시스템. 단일 HTML 파일 + Supabase로 동작하며, 관리자·학생·학부모 3역할을 지원합니다.

- **배포 주소:** https://nkmath01.github.io/nk/
- **데모 (설치 불필요):** https://nkmath01.github.io/nk/?demo=1

## 주요 기능

- 관리자 / 학생 / 학부모 3역할 분리 (전화번호 로그인)
- 주간테스트: 회차·문항 정의 + 학생×문항 채점 그리드(부분점수·오답원인 태그)
- 취약 진단: 단원×사고과정 히트맵, 우선 보완 순위, 오답 원인 패턴
- 과제 점검(풀이 채점 결과·걸린 시간), 첨삭 관리(루브릭)
- 합격 가능성(준비도 밴드), 주간 리포트(인쇄/PDF), 상담 기록(관리자 전용)
- 데모 모드(`?demo=1`) 내장 — 내장 샘플 데이터로 즉시 미리보기

## 구성

| 파일 | 용도 |
|------|------|
| `index.html` | 앱 전체 (브라우저에서 실행) |
| `supabase_setup.sql` | Supabase 테이블 생성 + 대학 시드 스크립트 |
| `README_설치가이드.md` | 원장·운영자용 설치·사용 가이드 |
